# 2026 프라이스 테이블 (최종 내부용 v1) — 권위 있는 재시드 원본

> 출처: 사용자 제공 스크린샷 2장 + `용역단가표.pdf`(2026 개편안). 본 파일이 `price_items` 재시드의 **단일 진실원천**이다.
> 수수료율: 카테고리 규칙 — **밴드 / 밴드(플레디스) = 20%, 그 외 전부 = 30%** (PDF 헤더·계산과 일치).
> 작가 실수령액 = 희망청구가 − 수수료 = 희망청구가 × (1 − fee). 앱에서 자동계산하므로 시드에는 넣지 않는다.
> 수식형(`is_formula=true`, `billing_price=NULL`): 희망청구가가 "청구가 공개" · "비용 논의 필요" · "-" 인 행. `formula_note`에 원문 보존.

표기: `작업내역 | 희망청구가 | (참고)수수료 | (참고)실수령` — 수수료·실수령은 검산용 참고값(시드 미포함).

## 카테고리 상수 정합 메모
- 기존 `CATEGORIES`(price-table page) / `REVENUE_CATEGORIES`(aggregator)와 **6개 모두 일치**: 앨범 / 방송·공연·시상식 / 광고 / 기타 / 밴드 / 밴드(플레디스). → 신규 카테고리 추가 불필요.
- 밴드 계열(밴드·밴드(플레디스))만 fee 0.20, 나머지 0.30 — `feeRateForCategory()`로 단일화.
- 총 행 수: 앨범 22 + 방송·공연·시상식 21 + 광고 5 + 기타 7 + 밴드(플레디스) 16 + 밴드 18 = **89행**.
